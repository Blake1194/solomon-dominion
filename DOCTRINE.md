# Solomon Doctrine — The 8 Cheatcodes

Earned in the school of hard knocks on 2026-05-07. Inviolable for every build.

## 1. Purple + gold from the ground up

Every surface, every layer, every brand touchpoint = the locked palette. No exceptions. See `theme/tokens.css`.

## 2. Mobile-first from the foundation

Touch targets ≥44×44px. Single-column default. Bottom nav (thumb zone). Voice button is hero. Tested on Galaxy S24 first. Desktop = scaled-up phone, never the inverse.

## 3. King of Customs — no third-party widgets visible

Embedded vendor widgets, "Powered by X" tags, third-party UI bleeding through — all replaced with custom UI hitting their APIs directly. Solomon's surface is 100% Solomon. Vendors live in the basement.

## 4. Sideload + self-host

`solomondominion.com` is the TRUE home. No Play Store. No third-party app stores. Sideload APK signed by us. Self-hosted release page + signed manifest + auto-updater. Owned domain, owned distribution, owned signing keys.

## 5. Tap-to-open, never paths in chat

Files share via attachments rendered as tappable cards. No `computer://` links, no Windows paths in messages, no copy-paste-this-into-your-browser. Mobile-first means tap-to-open from anywhere.

## 6. DO! is extractor only

DO! is a Chrome side-panel browser AI extension. One-shot tasks. Manual invocation by Blake. Output is research/audit text Blake relays to Claude. NOT autonomous, NOT scheduling its own work. Bridge 7 (Probes → QC) is built BY Claude USING DO!'s output, not BY DO! itself.

## 7. Final Mile Resistance is real

Friction always spikes at the end of a project. Plan for it: surface known friction points BEFORE they hit, batch Blake-only steps so they're done in one sitting, never make him do auth flows on phone.

## 8. No Linux user-facing complexity

Blake never touches Docker, terminals, ssh, or Linux config files. Cloud-hosted APIs only. If Linux runs underneath the stack, it stays invisible behind a URL endpoint. No "open a terminal and run this command" workflows. Ever.

---

**These are the kingdom's laws. Phase 1 honors all 8 from line one.**
